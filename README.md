# FlappyBird
Flappy Bird **Lumierea** Ver.  
**Features :**  
Left Click or Spacebar to control player  
Pause & Resume  
Score  
Highscore  
Reset HighScore  
you will get different medal if you reach specific score :  
-Bronze Medal >= 5 Score  
-Silver Medal >= 10 Score  
-Shiny Silver Medal >= 15 Score  
-Gold Medal >= 20 Score  
Enjoy!
